# sky-kanri

チームの進行を整理するための、軽量なタスク・プロジェクト管理ツール。
ブラウザだけで動き、データはお使いの端末（LocalStorage）にのみ保存されます。

公開デモ: https://playmark0227-svg.github.io/sky-kanri/

## できること

- カンバンボード（リスト・カードのドラッグ&ドロップ移動）
- タスクの詳細管理（説明はマークダウン、担当者複数、期間、ラベル、チェックリスト、コメント、フォロー、添付）
- ガントチャート（プロジェクト別 / 全体横断、定休日と祝日の反映）
- ダッシュボード（マイタスク、自分宛コメント、お知らせ、ブックマーク、進捗）
- 全タスクの横断ビュー（フィルタ、ソート、CSVエクスポート）
- 検索（タスク名・説明・コメント横断）
- アーカイブと復元
- ステータス自動変更（リスト移動でステータスが切り替わる）
- 休日設定（曜日定休、日本の祝日、個別休業日）
- ユーザー権限（組織5段階、プロジェクト4段階）
- CSV取り込み
- Webhook と APIキー管理（設定UIのみ／デモ環境では実際の通信は行いません）
- ダークモード

## 開発

依存関係は CDN から読み込んでいるため、ビルドは不要です。
`index.html` をそのまま開くか、任意の静的サーバーで配信してください。

```sh
# 例
python3 -m http.server 8000
```
