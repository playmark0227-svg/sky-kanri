# sky-kanri

Jootoスタイルのタスク・プロジェクト管理ツールのシングルHTMLクローン。

## 機能

- カンバンボード（リスト/カード、ドラッグ&ドロップ）
- タスク詳細（マークダウン説明、担当者、期限、ラベル、チェックリスト、コメント、ステータス、フォロー、ファイル添付）
- ガントチャート（休日反映：定休日/日本祝日/カスタム）
- ダッシュボード（マイタスク／メンション／掲示板／共有ブックマーク／フォロー中／プロジェクト進捗）
- 全てのタスク（横断フィルタ・ソート・CSVエクスポート）
- 全てのガントチャート
- 検索（タスク・コメント横断）
- アーカイブ（プロジェクト・タスク）
- 行動履歴
- ステータス自動変更（リスト→ステータスのマッピング）
- 休日設定
- ラベル管理（最大50個、色変更）
- ユーザー権限（組織5段階＋プロジェクト4段階）
- メンバー管理
- CSVインポート/エクスポート
- AIタスク生成β版
- AIタスク診断（要約粒度1-5、フィードバック口調1-5）
- API & Webhook 管理
- 掲示板・共有ブックマーク
- 通知センター
- マークダウン対応
- 一括選択・操作
- ダークモード

## 起動方法

`index.html` をブラウザで開くだけ。データはブラウザのLocalStorageに自動保存されます。
